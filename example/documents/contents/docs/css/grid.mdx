### start

アイテムを先頭に寄せる

### center

アイテムを中央に寄せる

### end

アイテムを末尾に寄せる

### space-between

各アイテムを均等に配置し最初のアイテムは先頭に寄せ、最後のアイテムは末尾に寄せる

### space-around

各アイテムを均等に配置し各アイテムの両側に半分の大きさの間隔を置く

### space-evenly

各アイテムを均等に配置する
